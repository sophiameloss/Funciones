etiquetas extras

Etiqueta pre

La etiqueta pre sirve para respetar espacios y líneas en blanco, al contrario de la etiqueta párrafo

Hola soy Sophia tengo 16 años 

        Hola   soy  Sophia
        tengo  16   años

Etiqueta abr

La etiqueta abr se utiliza para las abreviaturas

Según datos de la OMS el consumo excesivo de azucar provoca daños severos en la salud de los niños

Etiqueta bdo

Si se desea cambiar la dirección de un texto utiliza la etiqueta bdo

Este es un texto normal

Este es un texto diferente

Etiqueta cite

La etiqueta cite se utiliza para escribir citas textuales

Your comfort zone will kill you por: Pinterest

Etiqueta Small

La etiqueta small hace el texto más pequeño

texto esta dentro de una etiqueta small

Etiqueta sub

Si se desea escribir un subíndice, se debe utilizar la etiqueta sub

1566

Etiqueta sup

Si se desea escribir un superíndice se debe utilizar la etiqueta sup

1669

Etiqueta time

Para escribir fechas y horas se debe utilizar la etiqueta time

Mañana es día 2024-11-06

Mañana es día 6 de noviembre de 2024

Etiqueta adress

Para especificar al navegador que el texto se refiere a una dirección se utiliza la etiqueta adress

Loma dorada No. 100 Colonia Lomas de Rosales CP 89100 Tampico, Tams.
